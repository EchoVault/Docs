# ZRANDMEMBER

<Page
  syntax={`ZRANDMEMBER key [count [WITHSCORES]]`}
  module={"sortedset"}
  categories={["sortedset", "read", "slow"]}
  description={`Return a list of length equivalent to count containing random members of the sorted set.
If count is negative, repeated elements are allowed. If count is positive, the returned elements will be distinct.
WITHSCORES modifies the result to include scores in the result.`}

  examples={{

      },
      {
        description: "Get 4 non-unique random members from the sorted set with scores:",
        code: `ZRANDMEMBER key -4 WITHSCORES`
